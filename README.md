# 중1 수학 증명 학습 데모

중학교 1학년 수학 교과서 전 단원 증명 18개를 3-Phase(순서 맞추기 → 빈칸 채우기 → 백지 증명) 방식으로 학습할 수 있는 인터랙티브 목업.

🌐 **데모**: https://sigongjoa.github.io/proof-m1-demo/

## 📚 포함된 증명 (18개)

| 단원 | 증명 | 도형 |
|---|---|---|
| **평면도형 (7)** | 삼각형 내각의 합 / 맞꼭지각 / 평행선 동위각·엇각 / 외각 / n각형 내각합 / 외각합 360° / 정n각형 한 내각 | ✅ 5개 SVG |
| **수와 연산 (4)** | 소인수분해 유일성 / 짝수+짝수 / (−)×(−) / GCD×LCM=ab | — |
| **문자와 식 (4)** | 지수법칙 곱셈 / 거듭제곱 / 분배법칙 / 동류항 정리 | — |
| **방정식 (2)** | 등식의 성질·이항 / 일차방정식 해의 유일성 | — |
| **그래프 (1)** | 정비례 y=ax 직선 | — |

## 🎮 학습 플로우

1. **🔵 Phase 1 — 순서 맞추기**: 두 카드를 차례로 탭하거나 ▲▼ 버튼으로 순서 바꾸기
2. **🟢 Phase 2 — 빈칸 채우기**: 보기에서 골라 빈칸을 채우는 객관식
3. **🔴 Phase 3 — 백지 증명**: 자유 서술 → 교사 검토 → SRS 박스 승급

## 🧱 기술 특징

- 외부 의존성 **없음** — 순수 HTML/CSS/JS 단일 파일
- 빌드 과정 불필요
- 모바일 반응형
- GitHub Pages 자동 배포

## 🔧 로컬 실행

```bash
# 그냥 브라우저로 index.html 열거나
python3 -m http.server 8000
# → http://localhost:8000
```

## 📐 도형 자동 검증

`삼각형 내각의 합` 등 도형이 포함된 증명은 SVG 좌표로 기하 불변량(평행성, 각도, 엇각)을 프로그래매틱 검증.
잘못된 좌표는 빌드 단계에서 거부됨.
